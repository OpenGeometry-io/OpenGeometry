## Install OpenGeometry

Install the OpenGeometry package using your preferred package manager:

<CodeGroup>

```bash npm
npm install opengeometry
```

```bash yarn
yarn add opengeometry
```

```bash pnpm
pnpm add opengeometry
```

</CodeGroup>

## Peer Dependencies

OpenGeometry requires Three.js for 3D rendering. Install it as a peer dependency:

<CodeGroup>

```bash npm
npm install three@^0.168.0
```

```bash yarn
yarn add three@^0.168.0
```

```bash pnpm
pnpm add three@^0.168.0
```

</CodeGroup>

## TypeScript Support

If you're using TypeScript, install the Three.js type definitions:

<CodeGroup>

```bash npm
npm install --save-dev @types/three
```

```bash yarn
yarn add --dev @types/three
```

```bash pnpm
pnpm add --dev @types/three
```

</CodeGroup>

OpenGeometry includes built-in TypeScript definitions, so no additional types are needed for the core library.

## WebAssembly Setup

OpenGeometry uses WebAssembly for high-performance geometry. You need to do two things:

1. Serve the `.wasm` file so your app can fetch it.
2. Call `await OpenGeometry.create({ wasmURL })` once before you create primitives or shapes.

### Where the WASM file lives

When installed from npm, the WASM binary is located at:

`node_modules/opengeometry/opengeometry_bg.wasm`

### Vite (recommended)

Copy the WASM file into your app `public/` folder so it is served at `/opengeometry_bg.wasm`:

```bash
cp node_modules/opengeometry/opengeometry_bg.wasm public/opengeometry_bg.wasm
```

Then initialize OpenGeometry like this:

```ts
await OpenGeometry.create({ wasmURL: "/opengeometry_bg.wasm" });
```

### Other setups

As long as the WASM file is served and you pass the correct `wasmURL`, OpenGeometry works. If your
build fingerprints assets, pass the fingerprinted URL.

## Verify Installation

Create a simple test file to verify your installation:

```typescript
import { OpenGeometry, Cuboid, Vector3 } from 'opengeometry';
import * as THREE from 'three';

// Initialize the WASM module once before creating shapes
await OpenGeometry.create({ wasmURL: '/opengeometry_bg.wasm' });

// Create a simple test cuboid
const cuboid = new Cuboid({
  center: new Vector3(0, 0, 0),
  width: 2,
  height: 1,
  depth: 1,
  color: 0x4460ff
});

console.log('OpenGeometry cuboid created:', cuboid);
console.log('OGID:', cuboid.ogid);
```

<Note>
  Call `await OpenGeometry.create(...)` once before constructing kernel-backed primitives and shapes.
</Note>
