## Overview

`Cylinder` creates a cylindrical solid aligned to the Y axis. It supports configurable segment
count, partial-angle sweeps, kernel-backed placement, freeform conversion, parsed BRep access, and
boolean subtraction.

## Constructor

```ts
const cylinder = new Cylinder(options?: ICylinderOptions);
```

### ICylinderOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the cylinder. Auto-generated when omitted.
</ParamField>

<ParamField path="center" type="Vector3" required>
  Center point of the cylinder.
</ParamField>

<ParamField path="radius" type="number" required>
  Cylinder radius.
</ParamField>

<ParamField path="height" type="number" required>
  Cylinder height along the Y axis.
</ParamField>

<ParamField path="segments" type="number" required>
  Number of segments around the circumference.
</ParamField>

<ParamField path="angle" type="number" required>
  Sweep angle in radians. Use `2 * Math.PI` for a full cylinder.
</ParamField>

<ParamField path="color" type="number" required>
  Mesh color as a hex number.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Enables wide outline rendering with `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width used when `fatOutlines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig(), getConfig()

Updates or reads the current cylinder config.

```ts
cylinder.setConfig(options: CylinderConfigUpdate): void
const config = cylinder.getConfig();
```

### getPlacement(), getAnchor()

Returns cloned placement values or the current kernel anchor.

```ts
const placement = cylinder.getPlacement();
const anchor = cylinder.getAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the cylinder through the kernel transform path.

```ts
cylinder.setPlacement({ translation?, rotation?, scale? });
```

### toFreeform()

Converts the cylinder into a freeform geometry wrapper.

```ts
const freeform = cylinder.toFreeform(id?);
```

### getBrep()

Returns the parsed BRep object for the current cylinder.

```ts
const brep = cylinder.getBrep();
```

### subtract()

Subtracts another boolean operand from the cylinder and returns a renderable boolean result.

```ts
const result = cylinder.subtract(operand, options?);
```

### generateGeometry(), discardGeometry()

Rebuilds or disposes the current Three.js geometry.

```ts
cylinder.generateGeometry(): void
cylinder.discardGeometry(): void
```

## Examples

### Create a full cylinder

```ts
import { Cylinder, Vector3 } from "opengeometry";

const cylinder = new Cylinder({
  center: new Vector3(0, 1, 0),
  radius: 0.9,
  height: 2,
  segments: 48,
  angle: 2 * Math.PI,
  color: 0xf97316,
});
```

### Create a partial cylinder

```ts
const partial = new Cylinder({
  center: new Vector3(0, 1, 0),
  radius: 1.4,
  height: 2.2,
  segments: 48,
  angle: Math.PI * 1.25,
  color: 0x2563eb,
});
```

### Reconfigure and move the cylinder

```ts
cylinder.setConfig({
  radius: 1.2,
  height: 2.6,
  segments: 64,
});

cylinder.setTranslation(new Vector3(1.8, 0, 0));
```
