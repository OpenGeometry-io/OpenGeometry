## Overview

`Polygon` is the planar profile wrapper in the Three.js API. It creates a filled face on the XZ
plane, supports holes, can participate in coplanar booleans, converts to freeform geometry, and
extrudes into a renderable [`Solid`](/api/shapes/solid).

Outer loops should be provided counter-clockwise when viewed from above. Hole loops should be
provided clockwise.

## Constructor

```ts
const polygon = new Polygon(options?: IPolygonOptions);
```

### IPolygonOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the polygon. Auto-generated when omitted.
</ParamField>

<ParamField path="vertices" type="Vector3[]" required>
  Vertices for the outer loop.
</ParamField>

<ParamField path="holes" type="Vector3[][]" optional>
  Optional inner loops. Each hole is its own vertex array.
</ParamField>

<ParamField path="color" type="number" required>
  Face color as a hex number.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Enables wide outline rendering with `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width used when `fatOutlines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig(), getConfig()

Updates or reads the current polygon config.

```ts
polygon.setConfig(options: PolygonConfigUpdate): void
const config = polygon.getConfig();
```

### addVertices(), addHole()

Convenience helpers for replacing the outer loop or appending a new hole loop.

```ts
polygon.addVertices(vertices: Vector3[]): void
polygon.addHole(holeVertices: Vector3[]): void
```

### getPlacement(), getAnchor(), setAnchor(), resetAnchor()

Returns cloned placement values or reads/updates the current kernel anchor.

```ts
const placement = polygon.getPlacement();
const anchor = polygon.getAnchor();
polygon.setAnchor(anchor);
polygon.resetAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the polygon through the kernel transform path.

```ts
polygon.setPlacement({ translation?, rotation?, scale? });
```

### extrude()

Extrudes the polygon face and returns a renderable [`Solid`](/api/shapes/solid).

```ts
const solid = polygon.extrude(height);
```

<ParamField path="height" type="number" required>
  Finite, non-zero extrusion height.
</ParamField>

### toFreeform()

Converts the polygon into a freeform geometry wrapper.

```ts
const freeform = polygon.toFreeform(id?);
```

### getBrepData()

Returns the serialized BRep JSON string for the current polygon face.

```ts
const brepSerialized = polygon.getBrepData();
```

### subtract()

Subtracts another boolean operand from the polygon and returns a renderable boolean result.

```ts
const result = polygon.subtract(operand, options?);
```

### generateGeometry(), cleanGeometry(), dispose()

Rebuilds the polygon mesh or clears owned Three.js resources.

```ts
polygon.generateGeometry(): void
polygon.cleanGeometry(): void
polygon.dispose(): void
```

## Examples

### Polygon with a hole

```ts
import { Polygon, Vector3 } from "opengeometry";

const slab = new Polygon({
  vertices: [
    new Vector3(0, 0, 0),
    new Vector3(8, 0, 0),
    new Vector3(8, 0, 6),
    new Vector3(0, 0, 6),
  ],
  holes: [[
    new Vector3(5, 0, 4),
    new Vector3(5, 0, 2),
    new Vector3(3, 0, 2),
    new Vector3(3, 0, 4),
  ]],
  color: 0x2563eb,
});

slab.outline = true;
```

### Build a polygon incrementally

```ts
import { Polygon, Vector3 } from "opengeometry";

const footprint = new Polygon({
  vertices: [
    new Vector3(0, 0, 0),
    new Vector3(4, 0, 0),
    new Vector3(4, 0, 3),
    new Vector3(0, 0, 3),
  ],
  color: 0x0f172a,
});

footprint.addHole([
  new Vector3(2.8, 0, 2.2),
  new Vector3(2.8, 0, 1.2),
  new Vector3(1.8, 0, 1.2),
  new Vector3(1.8, 0, 2.2),
]);
```

### Extrude and cut an opening

```ts
import { Polygon, Vector3 } from "opengeometry";

const wallProfile = new Polygon({
  vertices: [
    new Vector3(-2.2, 0, -0.18),
    new Vector3(2.2, 0, -0.18),
    new Vector3(2.2, 0, 0.18),
    new Vector3(-2.2, 0, 0.18),
  ],
  color: 0x60a5fa,
});

const openingProfile = new Polygon({
  vertices: [
    new Vector3(-0.7, 0, -0.34),
    new Vector3(0.9, 0, -0.34),
    new Vector3(0.9, 0, 0.34),
    new Vector3(-0.7, 0, 0.34),
  ],
  color: 0xf97316,
});

const wall = wallProfile.extrude(2.8);
const opening = openingProfile.extrude(1.35);
opening.setTranslation(new Vector3(0, 0.85, 0));

const cut = wall.subtract(opening, {
  outline: true,
  fatOutlines: true,
  outlineWidth: 2,
  kernel: { mergeCoplanarFaces: true },
});
```

### Move or rotate the polygon profile

```ts
wallProfile.setPlacement({
  translation: new Vector3(1.5, 0, 0),
  rotation: new Vector3(0, Math.PI / 8, 0),
  scale: new Vector3(1, 1, 1),
});
```
