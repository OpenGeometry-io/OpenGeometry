## Overview

`Polygon` is the main planar profile wrapper in the Three.js API. It creates a triangulated face on
the XZ plane, supports holes, can participate in coplanar booleans, converts to freeform geometry,
and can be extruded into a renderable [`Solid`](/api/shapes/solid).

Outer vertices should be provided in counter-clockwise order when viewed from above. Hole loops
should be provided in clockwise order.

## Constructor

```ts
const polygon = new Polygon(options?: IPolygonOptions);
```

### IPolygonOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the polygon. Auto-generated when omitted.
</ParamField>

<ParamField path="vertices" type="Vector3[]" required>
  Vertices for the outer loop of the polygon.
</ParamField>

<ParamField path="holes" type="Vector3[][]" optional>
  Optional inner loops. Each hole is its own vertex array.
</ParamField>

<ParamField path="color" type="number" required>
  Fill color for the polygon face.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Use wide outlines backed by Three.js `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width for fat outlines. Values are sanitized and must be finite and greater than zero.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial translation for the polygon mesh.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial Euler rotation for the polygon mesh.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial scale for the polygon mesh.
</ParamField>

## Methods

### setConfig()

Updates polygon geometry or styling and regenerates the face when needed.

```ts
polygon.setConfig(options: PolygonConfigUpdate): void
```

### addVertices()

Replaces the outer loop with a new set of vertices.

```ts
polygon.addVertices(vertices: Vector3[]): void
```

### addHole()

Adds a new hole loop to the polygon.

```ts
polygon.addHole(holeVertices: Vector3[]): void
```

### extrude()

Extrudes the polygon face through the kernel BRep-face extrusion path and returns a renderable
[`Solid`](/api/shapes/solid).

```ts
const solid = polygon.extrude(height);
```

<ParamField path="height" type="number" required>
  Finite, non-zero extrusion height along the current kernel extrusion direction.
</ParamField>

### toFreeform()

Converts the polygon into a freeform geometry wrapper for direct editing workflows.

```ts
const freeform = polygon.toFreeform(id?);
```

### subtract()

Runs a coplanar boolean subtraction against another boolean operand.

```ts
const result = polygon.subtract(operand, options?);
```

### getBrepData()

Returns the serialized BRep JSON string for the polygon face.

```ts
polygon.getBrepData(): string | null
```

### dispose()

Disposes geometry, materials, and outline resources owned by the polygon.

```ts
polygon.dispose(): void
```

## Properties

</ResponseField>

  Gets or sets the outline color.
</ResponseField>

## Usage Examples

### Polygon with hole

```ts
import { Polygon, Vector3 } from "opengeometry";

const slab = new Polygon({
  vertices: [
    new Vector3(0, 0, 0),
    new Vector3(8, 0, 0),
    new Vector3(8, 0, 6),
    new Vector3(0, 0, 6),
  ],
  holes: [[
    new Vector3(5, 0, 4),
    new Vector3(5, 0, 2),
    new Vector3(3, 0, 2),
    new Vector3(3, 0, 4),
  ]],
  color: 0x2563eb,
});

slab.outline = true;
```

### Extrude and subtract an opening

```ts
import * as THREE from "three";
import { OpenGeometry, Polygon, Vector3 } from "opengeometry";

await OpenGeometry.create({ wasmURL: "/opengeometry_bg.wasm" });

const scene = new THREE.Scene();

const wallProfile = new Polygon({
  vertices: [
    new Vector3(-2.2, 0, -0.18),
    new Vector3(2.2, 0, -0.18),
    new Vector3(2.2, 0, 0.18),
    new Vector3(-2.2, 0, 0.18),
  ],
  color: 0x60a5fa,
});

const openingProfile = new Polygon({
  vertices: [
    new Vector3(-0.7, 0, -0.34),
    new Vector3(0.9, 0, -0.34),
    new Vector3(0.9, 0, 0.34),
    new Vector3(-0.7, 0, 0.34),
  ],
  color: 0xf97316,
});

const wall = wallProfile.extrude(2.8);
const opening = openingProfile.extrude(1.35);
opening.setTranslation(new Vector3(0, 0.85, 0));

const cut = wall.subtract(opening, {
  outline: true,
  fatOutlines: true,
  outlineWidth: 2,
  kernel: { mergeCoplanarFaces: true },
});

scene.add(cut);
```

## Best Practices

<Tip>
  Keep outer loops counter-clockwise and hole loops clockwise to match the kernel's face-building
  expectations.
</Tip>

<Info>
  Use `polygon.extrude(height)` when you want a proper renderable solid. Use `polygon.toFreeform()`
  when you want direct topology editing instead.
</Info>
