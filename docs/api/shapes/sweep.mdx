## Overview

`Sweep` creates a solid by sweeping a planar `profile` array along a 3D `path` array. It supports
optional start and end caps, kernel-backed placement, anchor control, freeform conversion, parsed
BRep access, and boolean subtraction.

<Note>
  Initialize `OpenGeometry.create(...)` before constructing `Vector3` or running the examples on
  this page.
</Note>

## Constructor

```ts
const sweep = new Sweep(options?: ISweepOptions);
```

### ISweepOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the sweep. Auto-generated when omitted.
</ParamField>

<ParamField path="path" type="Vector3[]" required>
  Path points used to drive the sweep. At least two points are required.
</ParamField>

<ParamField path="profile" type="Vector3[]" required>
  Profile points to sweep along the path. At least three points are required.
</ParamField>

<ParamField path="color" type="number" required>
  Mesh color as a hex number.
</ParamField>

<ParamField path="capStart" type="boolean" default="true">
  Adds a start cap when enabled.
</ParamField>

<ParamField path="capEnd" type="boolean" default="true">
  Adds an end cap when enabled.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Enables wide outline rendering with `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width used when `fatOutlines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig(), getConfig()

Updates or reads the current sweep config.

```ts
sweep.setConfig(options: SweepConfigUpdate): void
const config = sweep.getConfig();
```

### getPlacement(), getAnchor(), setAnchor(), resetAnchor()

Returns cloned placement values or reads/updates the current kernel anchor.

```ts
const placement = sweep.getPlacement();
const anchor = sweep.getAnchor();
sweep.setAnchor(anchor);
sweep.resetAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the sweep through the kernel transform path.

```ts
sweep.setPlacement({ translation?, rotation?, scale? });
```

### toFreeform()

Converts the sweep into a freeform geometry wrapper.

```ts
const freeform = sweep.toFreeform(id?);
```

### getBrep()

Returns the parsed BRep object for the current sweep.

```ts
const brep = sweep.getBrep();
```

### subtract()

Subtracts one or more boolean operands from the sweep and returns a renderable boolean result.

```ts
const result = sweep.subtract([operand], options?);
```

### generateGeometry(), discardGeometry()

Rebuilds or disposes the current Three.js geometry.

```ts
sweep.generateGeometry(): void
sweep.discardGeometry(): void
```

## Examples

### Create a capped sweep

```ts
import { Sweep, Vector3 } from "opengeometry";

const sweep = new Sweep({
  path: [
    new Vector3(-1.4, 0, -1.6),
    new Vector3(-0.5, 0.6, -0.5),
    new Vector3(0.4, 1.3, 0.8),
    new Vector3(1.2, 2.4, 2.1),
  ],
  profile: [
    new Vector3(-0.4, 0, -0.25),
    new Vector3(0.4, 0, -0.25),
    new Vector3(0.4, 0, 0.25),
    new Vector3(-0.4, 0, 0.25),
  ],
  color: 0x3b82f6,
  capStart: true,
  capEnd: true,
});
```

### Create an open-ended tube or rail

```ts
const openSweep = new Sweep({
  path: [
    new Vector3(0, 0, 0),
    new Vector3(0, 1.2, 0),
    new Vector3(1.4, 2.1, 0.6),
  ],
  profile: [
    new Vector3(-0.2, 0, -0.2),
    new Vector3(0.2, 0, -0.2),
    new Vector3(0.2, 0, 0.2),
    new Vector3(-0.2, 0, 0.2),
  ],
  color: 0x0f172a,
  capStart: false,
  capEnd: false,
});
```

### Update the path or profile

```ts
sweep.setConfig({
  path: [
    new Vector3(-1.6, 0, -1.8),
    new Vector3(-0.6, 0.8, -0.6),
    new Vector3(0.3, 1.6, 0.9),
    new Vector3(1.5, 2.7, 2.2),
  ],
});
```
