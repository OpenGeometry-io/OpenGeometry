## Overview

`Sphere` creates a spherical solid with configurable `widthSegments` and `heightSegments`. It
supports kernel-backed placement, freeform conversion, parsed BRep access, and boolean
subtraction.

## Constructor

```ts
const sphere = new Sphere(options?: ISphereOptions);
```

### ISphereOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the sphere. Auto-generated when omitted.
</ParamField>

<ParamField path="center" type="Vector3" required>
  Center point of the sphere.
</ParamField>

<ParamField path="radius" type="number" required>
  Sphere radius.
</ParamField>

<ParamField path="widthSegments" type="number" required>
  Horizontal tessellation count.
</ParamField>

<ParamField path="heightSegments" type="number" required>
  Vertical tessellation count.
</ParamField>

<ParamField path="color" type="number" required>
  Mesh color as a hex number.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Enables wide outline rendering with `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width used when `fatOutlines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig(), getConfig()

Updates or reads the current sphere config.

```ts
sphere.setConfig(options: SphereConfigUpdate): void
const config = sphere.getConfig();
```

### getPlacement(), getAnchor()

Returns cloned placement values or the current kernel anchor.

```ts
const placement = sphere.getPlacement();
const anchor = sphere.getAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the sphere through the kernel transform path.

```ts
sphere.setPlacement({ translation?, rotation?, scale? });
```

### toFreeform()

Converts the sphere into a freeform geometry wrapper.

```ts
const freeform = sphere.toFreeform(id?);
```

### getBrep()

Returns the parsed BRep object for the current sphere.

```ts
const brep = sphere.getBrep();
```

### subtract()

Subtracts another boolean operand from the sphere and returns a renderable boolean result.

```ts
const result = sphere.subtract(operand, options?);
```

### generateGeometry(), discardGeometry()

Rebuilds or disposes the current Three.js geometry.

```ts
sphere.generateGeometry(): void
sphere.discardGeometry(): void
```

## Examples

### Create a sphere

```ts
import { Sphere, Vector3 } from "opengeometry";

const sphere = new Sphere({
  center: new Vector3(0, 1.1, 0),
  radius: 1.1,
  widthSegments: 28,
  heightSegments: 18,
  color: 0x0891b2,
});
```

### Increase detail and move it

```ts
sphere.setConfig({
  radius: 1.4,
  widthSegments: 40,
  heightSegments: 28,
});

sphere.setPlacement({
  translation: new Vector3(2, 0.3, 0),
  rotation: new Vector3(0, 0, 0),
  scale: new Vector3(1, 1, 1),
});
```

### Convert to freeform for direct edits

```ts
const freeform = sphere.toFreeform();
```
