## Overview

`Wedge` creates a prism-like wedge solid defined by `center`, `width`, `height`, and `depth`. It
supports kernel-backed placement, freeform conversion, parsed BRep access, and boolean
subtraction.

## Constructor

```ts
const wedge = new Wedge(options?: IWedgeOptions);
```

### IWedgeOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the wedge. Auto-generated when omitted.
</ParamField>

<ParamField path="center" type="Vector3" required>
  Center point of the wedge.
</ParamField>

<ParamField path="width" type="number" required>
  Width along the X axis.
</ParamField>

<ParamField path="height" type="number" required>
  Height along the Y axis.
</ParamField>

<ParamField path="depth" type="number" required>
  Depth along the Z axis.
</ParamField>

<ParamField path="color" type="number" required>
  Mesh color as a hex number.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Enables wide outline rendering with `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width used when `fatOutlines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig(), getConfig()

Updates or reads the current wedge config.

```ts
wedge.setConfig(options: WedgeConfigUpdate): void
const config = wedge.getConfig();
```

### getPlacement(), getAnchor()

Returns cloned placement values or the current kernel anchor.

```ts
const placement = wedge.getPlacement();
const anchor = wedge.getAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the wedge through the kernel transform path.

```ts
wedge.setPlacement({ translation?, rotation?, scale? });
```

### toFreeform()

Converts the wedge into a freeform geometry wrapper.

```ts
const freeform = wedge.toFreeform(id?);
```

### getBrepData()

Returns the parsed BRep object for the current wedge.

```ts
const brep = wedge.getBrepData();
```

### subtract()

Subtracts another boolean operand from the wedge and returns a renderable boolean result.

```ts
const result = wedge.subtract(operand, options?);
```

### generateGeometry(), discardGeometry()

Rebuilds or disposes the current Three.js geometry.

```ts
wedge.generateGeometry(): void
wedge.discardGeometry(): void
```

## Examples

### Create a wedge or ramp

```ts
import { Wedge, Vector3 } from "opengeometry";

const wedge = new Wedge({
  center: new Vector3(0, 0.85, 0),
  width: 2.2,
  height: 1.7,
  depth: 1.4,
  color: 0x7c3aed,
});
```

### Place and rotate the wedge

```ts
wedge.setPlacement({
  translation: new Vector3(1.5, 0, 0),
  rotation: new Vector3(0, Math.PI / 2, 0),
  scale: new Vector3(1, 1, 1),
});
```

### Resize the wedge

```ts
wedge.setConfig({
  width: 3.2,
  height: 1.9,
  depth: 2.4,
  color: 0x2563eb,
});
```
