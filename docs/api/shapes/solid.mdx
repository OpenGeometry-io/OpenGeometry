## Overview

`Solid` is the generic BRep-backed mesh wrapper used when the kernel produces a solid that is not a
dedicated parametric primitive. The most common public path is:

```ts
const solid = polygon.extrude(height);
```

`Solid` supports kernel-backed placement, outline rendering, freeform conversion, parsed and
serialized BRep access, and boolean subtraction.

<Note>
  Initialize `OpenGeometry.create(...)` before constructing `Vector3` or running the examples on
  this page.
</Note>

## Constructor

```ts
const solid = new Solid(options: ISolidOptions);
```

### ISolidOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the solid. Auto-generated when omitted.
</ParamField>

<ParamField path="brep" type="FreeformSource" required>
  Serialized BRep JSON, parsed BRep-like data, or another wrapper exposing supported BRep accessors.
</ParamField>

<ParamField path="color" type="number" required>
  Mesh color as a hex number.
</ParamField>

<ParamField path="fatOutlines" type="boolean" default="false">
  Enables wide outline rendering with `LineSegments2`.
</ParamField>

<ParamField path="outlineWidth" type="number" default="1">
  Outline width used when `fatOutlines` is enabled.
</ParamField>

<ParamField path="anchor" type="Vector3" optional>
  Optional modeling anchor.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Static helper

### Solid.extrude()

Extrudes a face-like BRep source and immediately wraps it as a `Solid`.

```ts
const solid = Solid.extrude(source, height, options);
```

## Methods

### getPlacement(), getAnchor()

Returns cloned placement values or the current modeling anchor.

```ts
const placement = solid.getPlacement();
const anchor = solid.getAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale(), setAnchor()

Placement helpers for moving the solid through the wrapped freeform placement path.

```ts
solid.setPlacement({ translation?, rotation?, scale? });
solid.setAnchor(anchor);
```

### toFreeform()

Converts the solid back into a freeform geometry wrapper.

```ts
const freeform = solid.toFreeform(id?);
```

### getBrepSerialized(), getLocalBrepSerialized(), getBrepData()

Returns the world-space serialized BRep, the local-space serialized BRep, or the parsed world-space
BRep object.

```ts
const serialized = solid.getBrepSerialized();
const localSerialized = solid.getLocalBrepSerialized();
const brep = solid.getBrepData();
```

### subtract()

Subtracts one or more boolean operands from the solid and returns a renderable boolean result.

```ts
const result = solid.subtract([operand], options?);
```

### generateGeometry()

Rebuilds the visible mesh and outline state from the wrapped BRep/freeform data.

```ts
solid.generateGeometry(): void
```

## Examples

### Recommended path: polygon extrusion

```ts
import { Polygon, Vector3 } from "opengeometry";

const profile = new Polygon({
  vertices: [
    new Vector3(-1, 0, -0.2),
    new Vector3(1, 0, -0.2),
    new Vector3(1, 0, 0.2),
    new Vector3(-1, 0, 0.2),
  ],
  color: 0x2563eb,
});

const wall = profile.extrude(2.8);
wall.outline = true;
```

### Use the static extrusion helper

```ts
import { Polygon, Solid, Vector3 } from "opengeometry";

const profile = new Polygon({
  vertices: [
    new Vector3(-0.8, 0, -0.25),
    new Vector3(0.8, 0, -0.25),
    new Vector3(0.8, 0, 0.25),
    new Vector3(-0.8, 0, 0.25),
  ],
  color: 0x0f172a,
});

const solid = Solid.extrude(profile, 2.4, {
  color: 0x10b981,
  outlineWidth: 2,
});
```

### Wrap an existing BRep payload

```ts
import { Solid } from "opengeometry";

const solid = new Solid({
  brep: existingBrepSerialized,
  color: 0x10b981,
});
```

### Convert the solid to freeform

```ts
const freeform = solid.toFreeform();
```
