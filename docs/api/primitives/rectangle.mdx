## Overview

`Rectangle` is a closed line-based profile wrapper on the XZ plane. It supports wide-line
rendering, offset generation, anchor management, placement helpers, freeform conversion, and a
parsed BRep accessor.

### IRectangleOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the rectangle. Auto-generated when omitted.
</ParamField>

<ParamField path="center" type="Vector3" required>
  Center point of the rectangle.
</ParamField>

<ParamField path="width" type="number" required>
  Rectangle width along the X axis.
</ParamField>

<ParamField path="breadth" type="number" required>
  Rectangle breadth along the Z axis.
</ParamField>

<ParamField path="color" type="number" required>
  Rectangle color as a hex number.
</ParamField>

<ParamField path="fatLines" type="boolean" default="false">
  Enables wide-line rendering with `Line2`.
</ParamField>

<ParamField path="lineWidth" type="number" optional>
  Wide-line width used when `fatLines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig()

Updates geometry or render style.

```ts
rectangle.setConfig(options: RectangleConfigUpdate): void
```

### getConfig(), getPlacement()

Returns the current rectangle config or cloned placement values.

```ts
const config = rectangle.getConfig();
const placement = rectangle.getPlacement();
```

### getAnchor(), setAnchor(), resetAnchor()

Reads or updates the modeling anchor used by the kernel-backed primitive.

```ts
const anchor = rectangle.getAnchor();
rectangle.setAnchor(anchor);
rectangle.resetAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the rectangle through the kernel transform path.

```ts
rectangle.setPlacement({ translation?, rotation?, scale? });
```

### getOffset()

Returns an offset result for the current rectangle profile.

```ts
const offset = rectangle.getOffset(
  distance,
  acuteThresholdDegrees = 35,
  bevel = true
);
```

### toFreeform()

Converts the rectangle into a freeform geometry wrapper.

```ts
const freeform = rectangle.toFreeform(id?);
```

### getBrep()

Returns the parsed BRep object for the current rectangle.

```ts
const brep = rectangle.getBrep();
```

### discardGeometry()

Disposes the current Three.js geometry buffer.

```ts
rectangle.discardGeometry(): void
```

## Examples

### Create a rectangle profile

```ts
import { Polyline, Rectangle, Vector3 } from "opengeometry";

const rectangle = new Rectangle({
  center: new Vector3(0, 0, 0),
  width: 2.4,
  breadth: 1.6,
  color: 0x2563eb,
  fatLines: true,
  lineWidth: 8,
});
```

### Create an inset or outset profile

```ts
const room = new Rectangle({
  center: new Vector3(0, 0, 0),
  width: 6,
  breadth: 4,
  color: 0x0f172a,
});

const inset = room.getOffset(-0.2);

const finishLine = new Polyline({
  points: inset.points,
  color: 0x2563eb,
});
```

### Move and rotate the profile

```ts
rectangle.setPlacement({
  translation: new Vector3(0, 0, 1.2),
  rotation: new Vector3(0, Math.PI / 6, 0),
  scale: new Vector3(1, 1, 1),
});
```
