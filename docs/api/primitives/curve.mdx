## Overview

`Curve` is a line-based wrapper driven by a `controlPoints` array. It supports kernel-backed
placement, anchor management, freeform conversion, and offset generation for parallel guide paths.

<Note>
  Initialize `OpenGeometry.create(...)` before constructing `Vector3` or running the examples on
  this page.
</Note>

### ICurveOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the curve. Auto-generated when omitted.
</ParamField>

<ParamField path="controlPoints" type="Vector3[]" required>
  Ordered control points used to build the curve geometry.
</ParamField>

<ParamField path="color" type="number" required>
  Curve color as a hex number.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig()

Updates the control point array or render color.

```ts
curve.setConfig(options: CurveConfigUpdate): void
```

### getConfig(), getPlacement()

Returns the current curve config or cloned placement values.

```ts
const config = curve.getConfig();
const placement = curve.getPlacement();
```

### getAnchor(), setAnchor(), resetAnchor()

Reads or updates the modeling anchor used by the kernel-backed primitive.

```ts
const anchor = curve.getAnchor();
curve.setAnchor(anchor);
curve.resetAnchor();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the curve through the kernel transform path.

```ts
curve.setPlacement({ translation?, rotation?, scale? });
```

### getOffset()

Returns an offset result for the current curve.

```ts
const offset = curve.getOffset(
  distance,
  acuteThresholdDegrees = 35,
  bevel = true
);
```

### toFreeform()

Converts the curve into a freeform geometry wrapper.

```ts
const freeform = curve.toFreeform(id?);
```

## Examples

### Create a guide curve

```ts
import { Curve, Vector3 } from "opengeometry";

const curve = new Curve({
  controlPoints: [
    new Vector3(-2.5, 0, -1.5),
    new Vector3(-0.9, 0, 0.8),
    new Vector3(0.8, 0, 0.55),
    new Vector3(2.4, 0, 1.4),
  ],
  color: 0x16a34a,
});
```

### Offset a curve into a second guide

```ts
const centerline = new Curve({
  controlPoints: [
    new Vector3(-3, 0, -1.1),
    new Vector3(-1.2, 0, 0.7),
    new Vector3(0.8, 0, 0.2),
    new Vector3(2.9, 0, 1.4),
  ],
  color: 0x0f172a,
});

const offset = centerline.getOffset(0.35);

const outerGuide = new Curve({
  controlPoints: offset.points,
  color: 0x2563eb,
});
```

### Update the control points

```ts
curve.setConfig({
  controlPoints: [
    new Vector3(-2.5, 0, -1.4),
    new Vector3(-1.1, 0, 1.2),
    new Vector3(1.2, 0, 0.7),
    new Vector3(2.8, 0, 1.9),
  ],
});

curve.setTranslation(new Vector3(0, 0.15, 0));
```
