## Overview

`Line` is the simplest line-based wrapper in the Three.js API. It creates a straight segment between
two points, supports optional wide-line rendering, exposes placement helpers, and can generate
offset geometry for parallel construction workflows.

<Note>
  Initialize `OpenGeometry.create(...)` before constructing `Vector3` or running the examples on
  this page.
</Note>

### ILineOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the line. Auto-generated when omitted.
</ParamField>

<ParamField path="start" type="Vector3" required>
  Start point of the segment.
</ParamField>

<ParamField path="end" type="Vector3" required>
  End point of the segment.
</ParamField>

<ParamField path="color" type="number" required>
  Line color as a hex number.
</ParamField>

<ParamField path="fatLines" type="boolean" default="false">
  Enables wide-line rendering with `Line2`.
</ParamField>

<ParamField path="width" type="number" default="20">
  Wide-line width used when `fatLines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig()

Updates geometry or render style and regenerates the segment when needed.

```ts
line.setConfig(options: LineConfigUpdate): void
```

### getConfig()

Returns the current line options.

```ts
const config = line.getConfig();
```

### getPlacement()

Returns the current placement values as cloned vectors.

```ts
const placement = line.getPlacement();
```

### setPlacement()

Updates translation, rotation, or scale through the kernel-backed placement path.

```ts
line.setPlacement({ translation?, rotation?, scale? }): void
```

### setTransform(), setTranslation(), setRotation(), setScale()

Convenience placement helpers.

```ts
line.setTransform(translation, rotation, scale);
line.setTranslation(translation);
line.setRotation(rotation);
line.setScale(scale);
```

### getOffset()

Returns a parallel offset result for the current segment.

```ts
const offset = line.getOffset(
  distance,
  acuteThresholdDegrees = 35,
  bevel = true
);
```

### toFreeform()

Converts the line into a freeform geometry wrapper for direct editing.

```ts
const freeform = line.toFreeform(id?);
```

### discardGeometry()

Disposes the current Three.js geometry buffer. Use this only when you are handling lifecycle
manually.

```ts
line.discardGeometry(): void
```

## Examples

### Create a line

```ts
import { Line, Vector3 } from "opengeometry";

const line = new Line({
  start: new Vector3(-2, 0, -0.8),
  end: new Vector3(2, 0, 0.8),
  color: 0x111827,
  fatLines: true,
  width: 10,
});

line.setTranslation(new Vector3(0, 0.2, 0));
```

### Create a parallel guide

```ts
const baseLine = new Line({
  start: new Vector3(-2, 0, 0),
  end: new Vector3(2, 0, 0),
  color: 0x111827,
});

const offset = baseLine.getOffset(0.4);

const guideLine = new Line({
  start: offset.points[0],
  end: offset.points[1],
  color: 0x2563eb,
});
```

### Update endpoints after creation

```ts
line.setConfig({
  start: new Vector3(-3, 0, -0.6),
  end: new Vector3(3, 0, 1.1),
  color: 0xf97316,
});
```
