## Overview

`Polyline` creates an ordered chain of straight segments. It supports optional wide-line rendering,
incremental point editing with `addPoint(...)`, placement helpers, freeform conversion, and offset
generation. The wrapper also exposes whether the current path is closed through `isClosed`.

<Note>
  Initialize `OpenGeometry.create(...)` before constructing `Vector3` or running the examples on
  this page.
</Note>

### IPolylineOptions

<ParamField path="ogid" type="string" optional>
  Unique identifier for the polyline. Auto-generated when omitted.
</ParamField>

<ParamField path="points" type="Vector3[]" required>
  Ordered points that define the path.
</ParamField>

<ParamField path="color" type="number" required>
  Polyline color as a hex number.
</ParamField>

<ParamField path="fatLines" type="boolean" default="false">
  Enables wide-line rendering with `Line2`.
</ParamField>

<ParamField path="width" type="number" default="20">
  Wide-line width used when `fatLines` is enabled.
</ParamField>

<ParamField path="translation" type="Vector3" optional>
  Initial placement translation.
</ParamField>

<ParamField path="rotation" type="Vector3" optional>
  Initial placement rotation in radians.
</ParamField>

<ParamField path="scale" type="Vector3" optional>
  Initial placement scale.
</ParamField>

## Methods

### setConfig()

Updates the points array or render style.

```ts
polyline.setConfig(options: PolylineConfigUpdate): void
```

### addPoint()

Appends a point to the current path and regenerates geometry when possible.

```ts
polyline.addPoint(point: Vector3): void
```

### getConfig(), getPlacement()

Returns the current polyline config or cloned placement values.

```ts
const config = polyline.getConfig();
const placement = polyline.getPlacement();
```

### setPlacement(), setTransform(), setTranslation(), setRotation(), setScale()

Placement helpers for moving the polyline through the kernel transform path.

```ts
polyline.setPlacement({ translation?, rotation?, scale? });
```

### getOffset()

Returns an offset result for the current path.

```ts
const offset = polyline.getOffset(
  distance,
  acuteThresholdDegrees = 35,
  bevel = true
);
```

### toFreeform()

Converts the polyline into a freeform geometry wrapper.

```ts
const freeform = polyline.toFreeform(id?);
```

### discardGeometry()

Disposes the current Three.js geometry buffer.

```ts
polyline.discardGeometry(): void
```

## Examples

### Create a polyline

```ts
import { Polyline, Vector3 } from "opengeometry";

const polyline = new Polyline({
  points: [
    new Vector3(-2.5, 0, -1.7),
    new Vector3(-1.2, 0, -0.9),
    new Vector3(0.3, 0, 0.1),
    new Vector3(1.1, 0, 0.9),
    new Vector3(2.4, 0, 1.7),
  ],
  color: 0x0f172a,
  fatLines: true,
  width: 10,
});
```

### Append points incrementally

```ts
const path = new Polyline({
  points: [
    new Vector3(-2, 0, -1),
    new Vector3(-0.5, 0, -0.3),
  ],
  color: 0x111827,
});

path.addPoint(new Vector3(0.6, 0, 0.5));
path.addPoint(new Vector3(2.1, 0, 1.2));
```

### Offset a centerline

```ts
const offset = polyline.getOffset(0.3);

const wallGuide = new Polyline({
  points: offset.points,
  color: 0x2563eb,
});
```
