## Overview

OpenGeometry can export projected 2D scenes to vector PDF documents. The current public story is:

- use projection APIs to derive 2D output from B-Rep geometry
- use native Rust PDF export helpers when you need configurable output
- optionally use `OGSceneManager.projectToPDF(...)` as a native convenience path

<Warning>
PDF export is experimental. Output and APIs may change. If you hit an exporter limitation or
confusing output, reach out on [Discord](https://discord.gg/9wJpbfgGGA).
</Warning>

<Note>
PDF export is only available in native (non-WASM) builds. Browser/WASM builds do not include the
PDF writer.
</Note>

## Native scene-manager convenience API

In native builds, `OGSceneManager` exposes:

```ts
manager.projectToPDF(sceneId, cameraJson, hlrJson, "drawing.pdf");
```

This convenience method:

1. projects the stored scene to 2D
2. applies hidden line removal if requested
3. exports a PDF file using the default `PdfExportConfig`

If you need custom page size, margins, or titles, use the Rust PDF helpers directly.

## Example (native JavaScript / Node.js style)

```ts
import { Cuboid, OGSceneManager, OpenGeometry, Vector3 } from "opengeometry";

await OpenGeometry.create({ wasmURL: "/opengeometry_bg.wasm" });

function toBrepSerialized(source: unknown) {
  const value =
    source && typeof (source as { getBrepSerialized?: () => string }).getBrepSerialized === "function"
      ? (source as { getBrepSerialized: () => string }).getBrepSerialized()
      : source && typeof (source as { getBrepData?: () => unknown }).getBrepData === "function"
        ? (source as { getBrepData: () => unknown }).getBrepData()
        : source && typeof (source as { getBrep?: () => unknown }).getBrep === "function"
          ? (source as { getBrep: () => unknown }).getBrep()
          : source;

  return typeof value === "string" ? value : JSON.stringify(value);
}

const manager = new OGSceneManager();
const sceneId = manager.createScene("pdf export");

const cuboid = new Cuboid({
  center: new Vector3(0, 1.5, 0),
  width: 5,
  height: 3,
  depth: 2,
  color: 0x10b981,
});

manager.addBrepEntityToScene(sceneId, "box-1", "Cuboid", toBrepSerialized(cuboid));

const cameraJson = JSON.stringify({
  position: { x: 8, y: 6, z: 8 },
  target: { x: 0, y: 0, z: 0 },
  up: { x: 0, y: 1, z: 0 },
  near: 0.1,
  projection_mode: "Orthographic",
});

const hlrJson = JSON.stringify({
  hide_hidden_edges: true,
});

manager.projectToPDF(sceneId, cameraJson, hlrJson, "technical-drawing.pdf");
```

## Rust PDF API

When you need configurable export, the native Rust API is the current source of truth.

### PdfExportConfig

```rust
pub struct PdfExportConfig {
    pub page_width_mm: f64,
    pub page_height_mm: f64,
    pub margin_mm: f64,
    pub line_width_mm: f64,
    pub auto_fit: bool,
    pub title: Option<String>,
}
```

Default configuration:

```rust
PdfExportConfig {
    page_width_mm: 297.0,
    page_height_mm: 210.0,
    margin_mm: 10.0,
    line_width_mm: 0.25,
    auto_fit: true,
    title: None,
}
```

### Core Rust helpers

```rust
pub fn export_scene_to_pdf(scene: &Scene2D, file_path: &str) -> PdfExportResult<()>
pub fn export_scene_to_pdf_with_config(
    scene: &Scene2D,
    file_path: &str,
    config: &PdfExportConfig,
) -> PdfExportResult<()>
pub fn export_scene_to_pdf_bytes(
    scene: &Scene2D,
    config: &PdfExportConfig,
) -> PdfExportResult<Vec<u8>>
```

### Example (Rust)

```rust
use opengeometry::export::pdf::{export_scene_to_pdf_with_config, PdfExportConfig};
use opengeometry::export::projection::{project_brep_to_scene, CameraParameters, HlrOptions};

let scene = project_brep_to_scene(&brep, &CameraParameters::default(), &HlrOptions::default());

let config = PdfExportConfig {
    margin_mm: 15.0,
    line_width_mm: 0.35,
    title: Some("OpenGeometry Drawing".to_string()),
    ..PdfExportConfig::default()
};

export_scene_to_pdf_with_config(&scene, "drawing.pdf", &config)?;
```

## Notes

- Input projection units are meters. PDF output is in millimeters.
- `auto_fit: true` scales and centers the drawing on the page.
- The scene-manager JavaScript convenience path currently uses the default PDF config.
- If you need custom paper sizes or in-memory bytes output, use the Rust PDF helpers directly.
